🚀 FitME – Fitness Tracking App

FitME is a modern fitness tracking mobile application built using React Native (Expo) and Firebase. It helps users track workouts, monitor calories, and visualize progress with charts.

📱 Features

🔐 Demo Login System (No authentication required)

🏋️ Add & Track Workouts

🔥 Calorie Tracking

📊 7-Day Statistics Chart

📈 Progress Bar for Daily Goals

☁️ Firebase Firestore Integration

⚡ Fast & Responsive UI

🛠️ Tech Stack

Frontend: React Native (Expo)

Routing: Expo Router

Backend: Firebase Firestore

Charts: react-native-chart-kit

UI Components: React Native + react-native-progress
